# PTUT Prototype de l'application

## ✨Importation✨

- Git init et tout le bordel
- Ouvrez le dossier dans votre IDE pref alias VSCode
- Dans le terminal exécuter:
```sh
npm install
```
- Vous pouvez ensuite lancer l'application avec:
```sh
expo start
```
